# 📄 FormFlow: Interactive UI for Contact & Tasks

## 🚀 Project Overview

**FormFlow** is a simple and interactive web application built using **HTML, CSS, and JavaScript**. It combines two essential features:

* 📬 A **Contact Form** with validation
* ✅ A **To-Do List** for task management

This project is ideal for beginners to understand **DOM manipulation, form handling, and user interaction**.

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the web page
* **CSS3** – Styling and layout (Grid, responsiveness)
* **JavaScript (Vanilla JS)** – Functionality and interactivity

---

## ✨ Features

### 📬 Contact Form

* User input fields (Name, Email, Message)
* Input validation using JavaScript
* Error message display for invalid submissions

### ✅ To-Do List

* Add new tasks dynamically
* Display tasks in a list format
* Interactive UI for task management

---
